## Canva Setup (One Template → Three Platforms)

1. Go to canva.com → Create Design → Instagram Post (1080x1080)
2. Set background to #030213
3. Create 7-slide template:
   - Slide 1: Hook (big bold stat)
   - Slides 2-5: Data breakdown
   - Slide 6: Key takeaway
   - Slide 7: CTA (blue background, "Track on Healtho")
4. Save as a **Brand Kit** in Canva:
   - Colors: #8b5cf6, #030213, #0e0b1e, #ffffff, #94a3b8
   - Font: Montserrat (Bold for headlines, Regular for body)
5. Duplicate the template for each new post — just swap the text
